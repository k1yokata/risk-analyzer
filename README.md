# Risk Analyzer

Risk Analyzer: Streamlit-приложение для анализа коррупционных рисков в государственных закупках на основе данных ProZorro Public Procurement Dataset.

## Цель проекта

Разработать систему раннего предупреждения, которая выявляет тендеры с потенциально повышенным коррупционным риском на основе риск-индикаторов и расчёта итогового риск-балла.

## Основные возможности

- загрузка CSV-файла с данными тендеров;
- проверка структуры датасета;
- расчёт риск-балла;
- классификация тендеров по уровням риска;
- фильтрация по году, категории, методу закупки и стоимости;
- поиск по buyer_id и supplier_id;
- визуальная аналитика;
- таблица тендеров с высоким риском;
- анализ активированных риск-индикаторов.

## Технологии

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Pytest
